A 2D art generator, built with [Paper.js](https://paperjs.org), running on [Snowpack](https://www.snowpack.dev/).

## Development

Install dependencies:

```sh
yarn
```

Start the dev server:

```sh
yarn start

# or

yarn snowpack dev
```

Build for deploys:

```sh
yarn build

# or

yarn snowpack build
```
